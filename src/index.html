This page serves as the entry point for our user-friendly weather app. Built with JavaScript, HTML, CSS, Webpack, and a weather API, this app empowers you to effortlessly retrieve weather information for various locations.

Key Features:

  • Location Search: Enter a city name or zip code to find out the current weather conditions at your desired location.
  • Weather Data: The app displays essential weather details, including temperature, weather description, and an icon, providing a clear picture of the current weather.
  • Sleek Design: The HTML structure and CSS styles create a visually appealing and responsive user interface that adapts seamlessly to different screen sizes.

Getting Started:

  • Explore the Weather: Type your desired location (city name or zip code) in the search bar and click "Search."

Project Structure:

  • The HTML files define the overall layout and elements of the welcome page. JavaScript handles the logic for search functionality and interactivity.
  • CSS styles control the visual appearance and responsiveness of the interface. Webpack is employed for code bundling and optimization.
